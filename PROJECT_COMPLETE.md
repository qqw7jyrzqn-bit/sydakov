## 🎯 Функциональность

### Роли пользователей:
1. **Инженер ПТО** - создаёт и редактирует скважины (черновики)
2. **Начальник ПТО** - согласует, утверждает, полный доступ
3. **Внешний специалист** - readonly + комментарии
4. **Поставщик** - только технические задания (readonly)

### Маршрут согласования:
Черновик → На согласовании → Согласовано НПТО → Утверждено → В работе → Завершено → Архив

### Уведомления создаются при:
- Отправке на согласование
- Смене статуса
- Добавлении комментария

### Генерация отчётов:
- Word-документы (.docx) с данными скважины
- Таблица истории согласований
- Автоматическое скачивание

### Экспорт данных:
- Excel-файлы со списком скважин
- Все основные поля

## 📦 Технический стек

- **Backend:** Django 4.2 LTS, Python 3.11+
- **Database:** SQLite (по умолчанию)
- **Frontend:** Bootstrap 5.3, Vanilla JS
- **Документы:** python-docx, openpyxl
- **Формы:** django-crispy-forms + crispy-bootstrap5
- **Фильтры:** django-filter
- **API:** djangorestframework (готов к расширению)
- **Debug:** django-debug-toolbar
- **Extensions:** django-extensions
