## Навигация по новым функциям

### 📊 Аналитика
**Где:** Верхнее меню → **"Аналитика"** (иконка графика)

**Что там:**
- 12 статистических карточек
- 3 интерактивных графика (Chart.js)
- Топ-10 месторождений
- Топ-3 авторов

**Кнопки:**
- **"Отчёт по месторождению"** - детальная статистика по выбранному месторождению
- **"Отчёт по эффективности"** - производительность сотрудников

### 🔍 Расширенный поиск
**Где:** Верхнее меню → **"Поиск"** (иконка лупы)

**Фильтры:**
- Текстовый поиск (название/описание)
- Месторождение
- Статус (выбор из списка)
- Автор
- Глубина (от и до)
- Дата создания (диапазон)

**Как использовать:**
1. Заполните нужные поля
2. Нажмите **"Найти"**
3. Результаты отобразятся в таблице ниже
4. Кнопка **"Сбросить"** очищает все фильтры

## Административная панель (Django Admin)

### Доступ к новым моделям
URL: http://127.0.0.1:8080/admin/

**Новые разделы:**
- **Wells → Комментарии** - просмотр всех комментариев
- **Wells → Теги** - управление тегами с цветами
- **Wells → История изменений** - полный аудит действий
- **Wells → Дедлайны** - управление сроками
- **Wells → Вложения** - загруженные файлы

### Пример: Создание тега
1. Перейдите в `/admin/wells/tag/`
2. Нажмите **"Добавить тег"**
3. Заполните:
   - Название: `Срочно`
   - Цвет: `#ff0000` (красный)
   - Описание: `Высокоприоритетные скважины`
4. Сохраните

## Проблемы и решения

### Проблема: Графики не отображаются
**Решение:** Проверьте подключение к интернету (Chart.js загружается с CDN)

### Проблема: Ошибка 500 при открытии аналитики
**Решение:** 
```bash
# Проверьте миграции
python manage.py showmigrations
python manage.py migrate
```

### Проблема: Нет данных на графиках
**Решение:** Создайте тестовые скважины через интерфейс или admin
